> 
> **Structure:** `Main Category (Tier 1)` → `Sub-Categories (Tier 2)`

---

## 📋 Category Structure Overview

```
BUSINESS_TYPE
├── Main Category 1
│   ├── Sub Category 1.1
│   ├── Sub Category 1.2
│   └── ...
├── Main Category 2
│   ├── Sub Category 2.1
│   └── ...
└── ...
```
